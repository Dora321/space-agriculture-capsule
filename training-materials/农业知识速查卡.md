## 第二部分：氮磷钾（NPK）——评委必问

### 什么是氮磷钾？

| 元素 | 化学符号 | 作用 | 生活类比 | 什么时候需要 |
|------|----------|------|----------|--------------|
| **氮** | N | 让叶子长得又大又绿 | 像人长身体要吃肉 | 苗期 + 营养生长期 |
| **磷** | P | 促进开花和根系发育 | 像人要结婚生孩子 | 花期 |
| **钾** | K | 让果实又大又甜，增强抗病 | 像人运动后补电解质 | 果期 |

### 我们项目里的对应关系

**叶菜类（生菜、小白菜、菠菜、韭菜）**：
```
苗期：用氮肥（N）——"先长根"
营养期：用氮肥（N）——"长叶子"  
采收期：用复合肥（NPK）——"维持平衡"
```

**果菜类（番茄、辣椒、黄瓜、茄子）**：
```
苗期：用氮肥（N）——"先长棵"
营养期：用氮肥（N）——"壮枝叶"
花期：用磷钾肥（PK）——"促花保果"
果期：用钾肥（K）——"膨果增甜"
```

### 评委可能追问：为什么果菜花期要"控水减氮"？

**标准回答**（队员3）：
```
"这是农业上的'控水促花'原理：

如果花期还浇很多水、施很多氮肥，
植物会'贪青'——就是一直长叶子，不开花。

就像一个人如果天天吃大鱼大肉，
就不想干活了。适当'饿一饿'，
反而会促进开花结果。

我们项目里的做法是：
番茄花期，土壤阈值从35提高到40，
意味着'不那么干就不浇'，
浇水时间从12秒减到正常水平，
这样植物就会'觉得压力'，开始开花。"
```

### 评委可能问：你们怎么补光的？

**标准回答**（队员1）：
```
"我们用12V COB补光灯条，通过继电器控制。

当光照传感器检测到光照低于植物需要的最低值，
比如番茄光照低于50%，
系统会自动开灯补光。

补光时长根据缺少的日照时长计算：
如果今天只晒了4小时，番茄需要8小时，
就补2小时（缺4小时的一半）。

单次补光最长2分钟（安全限制），
防止灯条过热或者植物被烤伤。"
```

### 评委可能问：为什么补光时长是"缺少时长的一半"？

**标准回答**（队员3可以补）：
```
"因为人工补光的光照强度比太阳弱。

太阳光强度大约是10万勒克斯（lux），
我们的COB灯条在30厘米距离大约是5000勒克斯，
只有太阳的5%。

所以补光时间不能简单等于缺少的时间，
我们用一半作为估算值，
实际效果还需要根据植物反应调整。

在真正的太空舱里，会用专业的植物生长灯，
光照强度可以做到太阳的80-90%，
补光时间会更短。"
```

## 第五部分：温度知识

### 各植物的适宜温度

| 植物 | 最低温度 | 最适温度 | 最高温度 | 超过35℃会怎样 |
|------|----------|----------|----------|---------------|
| 生菜 | 5℃ | 15-20℃ | 25℃ | 抽薹（变苦） |
| 小白菜 | 5℃ | 18-25℃ | 30℃ | 品质下降 |
| 菠菜 | 5℃ | 15-20℃ | 25℃ | 抽薹变苦 |
| 韭菜 | 5℃ | 15-25℃ | 30℃ | 停止生长 |
| 番茄 | 10℃ | 20-28℃ | 35℃ | 落花落果 |
| 辣椒 | 12℃ | 20-30℃ | 35℃ | 落花落果 |
| 黄瓜 | 10℃ | 22-30℃ | 35℃ | 畸形果 |
| 茄子 | 12℃ | 22-30℃ | 35℃ | 落花落果 |

### 评委可能问：超过35℃为什么不浇水？

**标准回答**（队员1）：
```
"高温时浇水有两个问题：

第一，'闷根'——高温时土壤里水分蒸发快，
浇水后土壤变成蒸汽室，根部会缺氧腐烂。

第二，'叶面灼伤'——水滴留在叶面上，
像放大镜一样聚焦阳光，烧出斑点。

所以我们的系统在温度超过35℃时，
会跳过浇水动作，等温度降下来再浇。

如果土壤极度干燥（低于阈值-15），
就是温度高也会救命浇水，
因为干死比闷根更严重。"
```

## 第八部分：5个高频农业Q&A（必背）

### 🌾 Q-A1: "你们8种作物为什么选这8种？"

```
"我们选了4种叶菜和4种果菜，是经过考虑的：

叶菜类代表'快速实验'：
30-50天就能看到完整生长周期，
适合在太空舱做短期育种实验。

果菜类代表'完整研究'：
有花期和果期，生长阶段更复杂，
适合研究太空环境对生殖生长的影响。

两类对比，可以研究一个核心问题：
太空环境对'营养生长'和'生殖生长'
的影响是否不同？
这是太空育种的真正前沿问题。"
```

### 🌾 Q-A2: "你们的植物数据是哪里来的？靠谱吗？"

```
"我们的数据有两个来源：

第一，中国农科院的蔬菜种植标准，
这是官方的农业技术规程。

第二，NASA VEGGIE项目，
这是国际空间站上实际种过菜的参数。

两组数据交叉验证，确保可靠性。

当然，我们的系统是教育原型，
实际种植还需要根据环境校准。
但我们的控制逻辑——
根据生长阶段自动调整水肥策略——
这个设计思路是正确的。"
```

### 🌾 Q-A3: "太空种菜和地面种菜有什么不同？"

```
"最大的不同有三个：

第一，微重力。根不知道往哪长，
水也不会自然往下流。
所以太空种植用的是'毛细管基质'，
不是土壤，水靠毛细力往上吸。

第二，光照。空间站每90分钟绕地球一圈，
所以太空一天有16次日出日落。
植物会'晕'，需要人工控制光周期。

第三，水循环。太空里水非常珍贵，
宇航员连汗都要回收。
所以太空种菜的水必须是闭环的：
植物蒸腾→冷凝回收→再浇回去。

我们的项目模拟了第三点：
用土壤湿度传感器精确控制浇水，
避免浪费水。"
```

### 🌾 Q-A4: "为什么叶菜只要氮肥，果菜要三种肥？"

```
"因为我们吃的部分不同：

叶菜，我们吃的是叶子。
氮肥的作用就是让叶子长得又大又绿，
所以叶菜全程只需要氮肥就够了。

果菜，我们吃的是果实。
苗期要先长叶子（氮肥），
花期要促进开花（磷钾肥），
果期要让果实膨大变甜（钾肥）。

一句话总结：
吃叶子给氮肥，吃果实要分三步。"
```

### 🌾 Q-A5: "你们的系统能处理植物病虫害吗？"

```
"目前不能。我们系统关注的是环境控制——
浇水、补光、温度监控。

病虫害防治需要图像识别和专业知识，
超出了我们当前项目的范围。

但我们的架构可以扩展：
加一个摄像头模块，用计算机视觉
检测叶片异常，就能实现病害预警。

这是我们的下一步计划。"
```
