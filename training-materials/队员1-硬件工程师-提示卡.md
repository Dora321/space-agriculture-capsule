## 第一步：认识7个零件（3秒指认）

| 零件 | 记住这句话 | 实物特征 |
|------|------------|----------|
| ESP32开发板 | "这是大脑" | 最大的那块板子，有USB口 |
| 土壤湿度传感器 | "插土里的探头" | 两根金属针 |
| DHT11温湿度传感器 | "测温度和湿度的" | 白色小方块 |
| 继电器模块 | "电子开关" | 透明塑料盒，有螺丝接线柱 |
| OLED显示屏 | "三页状态屏" | 蓝色小屏幕，红/黄键翻页 |
| 水泵 | "抽水的" | 接软管的 |
| ADC模拟键盘 | "4键1脚" | 4个彩色按键+4个电阻 |

**训练方法**：
- 把7个零件（含ADC模拟键盘）摆在桌上，教练说名字，学生指认
- 反过来：学生拿零件，说名字
- 目标：3秒内反应

---

**OLED三页快速记忆**：

| 页码 | 显示内容 | 翻页键 | 一句话 |
|------|----------|--------|--------|
| 第1页 | 土壤湿度大字+作物阶段图标 | 红键← / 黄键→ | "湿度多少，一目了然" |
| 第2页 | 生长阶段+进度条+施肥/水分 | 红键← / 黄键→ | "现在该施什么肥" |
| 第3页 | 内存+Pi在线状态+上次动作原因 | 红键← / 黄键→ | "Pi在线吗？系统稳吗？" |

> 蓝键单击进入菜单系统（Plant Select / Set Day / Manual Ctrl / System Info）

## 第二步：认识5条连线（指出来）

```
土壤湿度传感器  →  插在ESP32的GPIO34（ADC）
DHT11           →  插在ESP32的GPIO4
继电器-水泵      →  插在ESP32的GPIO5
继电器-补光灯    →  插在ESP32的GPIO18
UART通信线       →  TX=GPIO17, RX=GPIO16, GND（接树莓派，三线交叉共地）
OLED显示屏(I2C) →  SDA=GPIO21, SCL=GPIO22
```

**不要求懂为什么，只要求能指出来。**

**训练方法**：
- 指着ESP32上的接口，学生说"这是插土壤的"
- 拿着传感器，学生说"插在GPIO34"

## 第三步：3个固定动作（练到肌肉记忆）

### 动作1：指认零件+翻页演示（配合自我介绍）

**话术**：
```
"大家好，我是这座育种站的总工程师，杨灏辰——舱是我们造的，也是我们修的。
我们的种植舱，ESP32这边140元，加树莓派约400元，全套540元。

这是ESP32主控板，这是土壤湿度传感器，
这是DHT11温湿度传感器，这是继电器模块。

我们的OLED是三页显示屏——
（按黄键）第一页，土壤湿度实时大字；
（按黄键）第二页，生长进度条和施肥建议；
（按黄键）第三页，剩余内存和Pi在线状态。

三个页面让舱的状态一目了然。"
```

**动作分解**：
1. 拿起ESP32 → "这是大脑"
2. 拿起土壤传感器 → "这是插土里的"
3. 拿起DHT11 → "这是测温湿度的"
4. 指继电器 → "这是电子开关"
5. 按黄键翻页3次 → 每页配一句话

**训练目标**：边说边做，不用思考

---

### 动作2：拔UART线演示Pi离线自治（核心演示）

**话术**：
```
"现在我演示ESP32的离线自治能力。
（拔掉UART连接线）

大家看OLED第三页，Pi状态变成了'PI:OFF'，
系统自动切换到本地规则，继续正常浇水。

（插回UART线）
UART恢复后，Pi重新在线，系统重新采纳Pi的建议。"
```

**动作分解**：
1. 说"现在我演示离线自治"（手放在UART线旁）
2. 拔掉UART线
3. 按黄键翻到第三页 → 指Pi状态位置 → "大家看，PI:OFF"
4. 停顿2秒（让评委看清屏幕切换）
5. 插回UART线
6. 等3秒 → 再指Pi状态 → "PI:OK，采纳Pi建议"

**⚠️ 关键**：
- 拔线前提醒评委"我要拔UART线了"
- 插回后等3秒，让超时检测完成
- 如果屏幕没反应，说"系统在检测中，马上好"

---

### 动作3：指系统架构图（配合陈美伊）

**话术**：
```
"这是我们系统的流程图：

左边是传感器采集数据，
中间是两层决策（AI→本地规则），
右边是执行器做出动作。

整个流程每60秒循环一次，实现全自动种植。"
```

**动作分解**：
1. 走到KT板前
2. 手指从左向右移动（传感器 → 决策 → 执行）
3. 在每个区域停留2秒
4. 最后指向"两层决策架构"

## 第四步：背3段话术

### 话术1：自我介绍+硬件介绍（约20秒）

```
"大家好，我是这座育种站的总工程师，杨灏辰。
舱是我们造的、也是我们修的，ESP32这边140元，加树莓派约400元，全套540元。

这是主控板、土壤传感器、DHT11、继电器。
OLED是三页屏——
第1页湿度，第2页生长进度，第3页Pi状态。"
```

**背诵技巧**：
- 分段背：自我介绍 → 成本 → 指零件 → OLED三页
- 边做动作边背

---

### 话术2：Pi离线自治演示（约20秒）

```
"现在我演示离线自治。
（拔UART线）

大家看第三页，PI:OFF，本地规则接管，浇水不停。
我造它的时候最在意这一点——人不在、网不在，菜也不能死。

（插回UART线）
UART恢复后Pi重新在线。"
```

**背诵技巧**：
- 关键动作：拔线 → 翻第三页 → 指PI:OFF → 插回 → 指PI:OK
- 练到"拔线的时候嘴巴自动说出下一句"
- 那句"人不在、网不在，菜也不能死"是总工程师的招牌话，记住它

---

### 话术3：容错机制（约20秒）

```
"我们有多层保护：
第一，看门狗，程序卡死自动重启；
第二，传感器失效用安全值；
第三，Pi离线自动回退本地规则。

两层决策：AI→本地规则，
AI不在线，ESP32自己判断。"
```

**背诵技巧**：
- 记住"两层数字"：1、2
- 记住关键词：看门狗、安全值、本地规则

## 第五步：可能的Q&A（3个问题）

### Q1: 你们的硬件成本真的只要ESP32这边140元，加树莓派约400元，全套540元吗？

**回答**：
```
"是的：ESP32 30元，传感器40元，
继电器和线材20元，水泵和灯带50元，总共ESP32这边140元，加树莓派约400元，全套540元。
用Arduino或树莓派会贵3-5倍。"
```

---

### Q2: ESP32能稳定运行吗？不会死机吗？

**回答**：
```
"我们有多层保护：
看门狗3秒自动重启；传感器失效切安全值；
Pi离线自动回退本地规则。
实机验收全部通过，系统可以长期运行。"
```

---

### Q6: OLED三页界面分别显示什么？

**回答**：
```
"第一页——土壤湿度大字，3米外也能看清；
第二页——生长进度条和施肥需求；
第三页——剩余内存和Pi在线状态。

三页设计的意义是：
太空舱里屏幕小，按功能分层展示，
宇航员快速定位信息，不用在拥挤的界面上找。"
```

**⚠️ 注意**：
- 如果评委想看，按黄键翻页演示
- 如果屏幕不亮，用截图代替
