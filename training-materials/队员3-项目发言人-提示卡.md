## 🎭 我的 2035 身份（开口前先记住）

> 比赛主题是《2035年的我》。**台上的我，是 2035 年长大后的自己——月宫太空育种站的农业育种专家。**
> 我决定种什么、为什么种；杨灏辰是总工程师（造舱），张煦涵是站长（管系统大脑）。
> 开场报身份，结尾讲"我为什么想做这件事"，中间技术内容不变。
>
> ⚠️ **核心立意（别讲偏）**：我们做的是**太空育种**——加快育种反馈周期，多快好省地为**全人类**选育更优良的种子。温情来自**育种人的奉献精神**（接袁隆平的棒），**不是**"在太空里吃口热菜"。

## 第一步：育种历史故事（5章，要能复述）

### 第1章：远古时代（公元前8000年）

**核心矛盾**：靠运气，周期极长

**记住这句话**：
```
"原始人采集野生小麦，掉在地上的种子第二年长出来。
他们发现：某些野生植株的籽粒不容易脱落，
于是开始有意识地留种。

但问题来了：完全靠运气，
一个有益性状的出现可能要等几百年。

从野生小麦到驯化小麦，用了8000年。"
```

**关键词记忆**：原始人 → 留种 → 靠运气 → 8000年

---

### 第2章：孟德尔（1865年）

**核心矛盾**：可以计划，但还是要种出来看

**记住这句话**：
```
"1865年，孟德尔用豌豆做实验，发现了遗传规律：
性状是由基因控制的，有显性/隐性之分。

这让育种从'靠运气'变成了'有计划'。
但问题还是：要种出来看结果，然后筛选。

周期：杂交到稳定需要5-10年。"
```

**关键词记忆**：孟德尔 → 遗传规律 → 有计划 → 5-10年

---

### 第3章：袁隆平（1973年）

**核心矛盾**：产量大幅提升，但周期还是长

**记住这句话**：
```
"1973年，袁隆平先生研发出杂交水稻，
比常规稻增产20-30%，解决了中国人的吃饭问题。

但杂交水稻的研发周期还是5-10年。
到今天，出一个新品种仍要8-12年。"
```

**关键词记忆**：袁隆平 → 杂交水稻 → 增产20-30% → 还是要8-12年

---

### 第4章：太空育种（1987年至今）

**核心矛盾**：诱变强，但反馈周期被"搭载机会"卡死

**记住这句话**：
```
"1987年，中国开始太空育种实验。
把种子送上太空，利用宇宙射线和微重力诱变。

太空诱变频率比地面高3-4倍。
中国已经有240多个太空育种品种通过审定。

但问题来了：
种子上天飞一圈，回地面还要种5-8代才能筛选。

反馈周期被'搭载机会'卡死了——
火箭床位有限，不能随时上去。"
```

**关键词记忆**：1987年 → 太空诱变 → 频率高3-4倍 → 但还要5-8代 → 搭载机会稀缺

---

### 第5章：我们的项目

**核心解决方案**：让种植和筛选发生在太空舱里

**记住这句话**：
```
"我们的解决方案是：
让种植和筛选本身就发生在太空舱里！

这样反馈周期可以砍掉一个量级：
不用等火箭搭载，不用回地面种5-8代，
在轨就能完成初筛。

今天它是一个教育原型，
明天它的控制设计可以直接迁移到在轨实验舱。

人类的育种史，就是一部缩短反馈周期的历史。
我们的项目，是这段历史的下一个章节。"
```

**关键词记忆**：让种植发生在太空舱里 → 反馈周期砍一个量级 → 教育原型 → 未来可迁移

## 第二步：答辩稿（约3分钟）

### 【0-35秒：立意】

```
"各位评委好。

现在是 2035 年。我是月宫太空育种站的农业育种专家，陈美伊。
（指两位队友）这是我们的总工程师，这是我们的育种站长。
今天请让 2035 年的我们，带你们看看我们每天的工作。

为什么是我们来做这件事？让我们看一段历史：

公元前8000年，人类驯化野生小麦，花了10000年；
1865年，孟德尔发现遗传规律，缩短到10年；
1973年，袁隆平杂交水稻，周期5-10年；
1987年，中国开始太空育种——
但种子上天绕一圈，回地面还要种5-8代才能筛选。

问题来了：太空诱变是地面的3-4倍，
但反馈周期被'搭载机会'卡死了。

所以 2035 年的我，想做的就是一件事：
让育种和选种，直接发生在太空舱里！
把出一个好品种的周期，多快好省地一次次缩短，
为全人类更快地育出更好的种子。

今天它是教育原型，
明天它的设计可以迁移到真实太空育种舱。"
```

- 分段背：我是谁 → 育种历史 → 问题 → 我的解决方案
- 每段练50遍

```
"最后，作为育种专家，我想说三件事：

第一，两层决策设计。

（放慢，看着评委）

为后人选出更好的种子，让更多人吃饱、吃好。
袁隆平爷爷一辈子的梦想，就是让所有人远离饥饿。
```

## 农业知识速记（评委问植物问题时的保命卡）

### 必须记住的口诀

```
氮长叶，磷开花，钾结果
叶菜三阶段：苗→长→收
果菜四阶段：苗→长→花→果
叶菜全程氮，果菜分三步
叶菜光照6-8，果菜光照8-12
高温不浇水（35℃），低温不动手（8℃）
```

### 被问到农业知识时的万能回答模板

```
"这个问题很好。在我们的plants.json里，
我们为8种作物建立了生长阶段模型。

（然后根据问题选一个补充）

- 如果问肥料：氮长叶磷开花钾结果，叶菜全程氮肥，果菜分三步。
- 如果问光照：叶菜6-8小时，果菜8-12小时。
- 如果问温度：超过35℃跳过浇水，低于8℃停止动作。
- 如果问数据来源：中国农科院 + NASA VEGGIE + AI交叉验证。
- 如果问不知道的：这个问题我们还没深入研究，
  作为教育原型，我们主要关注控制设计。
  谢谢评委老师，我们会继续学习。"
```
