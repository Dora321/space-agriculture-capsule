# 教练指南 · 7+2天零基础培训手册

> **适用对象**：Su（指导老师）
> **培训目标**：3个零基础学生，7-9天内能完成3分钟答辩 + Q&A
> **核心原则**：不追求理解原理，只追求能说、能演示

---

## 一、培训总览

### 9天节奏

```
第1-2天：分头学（每人学自己的部分）
第3-4天：背话术（每人背熟3段话）
第5-6天：合练（三人配合走流程）
第7天：联排（全要素模拟答辩）
第8-9天：新增专项（OLED翻页盲操、ADC键盘、双层架构话术）
```

## 二、第1-2天：分头学

### 队员1（第1-2天）

**教练动作**：
1. 把7个零件（含模拟键盘4键）摆在桌上
2. 指着零件说名字，让学生跟读3遍
3. 让学生自己拿零件说名字（不按顺序）
4. 认识接线：指着ESP32上的接口让学生说
5. **OLED三页翻页训练**（新增）：
   - 按红键→第1页（湿度大字+图标），让学生读出来
   - 按黄键→第2页（生长进度+肥料），让学生读出来
   - 再按黄键→第3页（内存+IP），让学生读出来
   - 按蓝键→进入菜单，红/黄键滚动，绿键确认，蓝键返回
   - 菜单包含：植物选择、天数设置、手动控制、系统信息
   - 天数设置：红/黄键±1，按住不放自动加速
   - 反复练10遍

**验收标准**：
- 7个零件3秒内指认正确
- 接线能指出来
- OLED三页翻页 + 菜单进入退出，不用思考就能完成

**常见问题**：
- 学生分不清DHT11和土壤传感器 → 用颜色区分（DHT11白色小方块、土壤传感器两根金属针）
- 学生拿电路板手抖 → 让学生多拿几次，习惯手感
- **OLED按键不灵敏**（新增） → 教学生"按下去听到滴答声才算"，不是轻轻碰
- **学生搞混红/黄/蓝键**（新增） → 口诀："红上黄下绿确定，蓝键回家（单击进菜单）"

---

### 队员2（第1-2天）

**教练动作**：
1. 用"定时浇水 vs AI浇水"的类比讲解
2. 打开Web大屏，让学生操作3次
3. 让学生指认3个区域（左/中/右）
4. 学生练习操作（打开/指认/刷新）
5. **双层架构理解**（新增）：
   - 用一句话讲："ESP32是驾驶员，树莓派是副驾驶——副驾驶可以休息，驾驶员不能停"
   - 让学生用自己的话复述一遍

**验收标准**：
- 能说清"本地规则 vs AI"的区别
- 能用自己的话说出3个AI触发条件
- 3个操作练到不用思考
- 能说出"ESP32飞控 + Pi载荷"的分工

**常见问题**：
- 学生不理解"AI决策" → 用生活类比："本地规则像闹钟定时响，AI像妈妈看天气决定要不要带伞"
- 学生操作电脑不熟练 → 提前设置好桌面快捷方式，双击就行
- 学生看不懂大屏数据 → 只记3个区域的名字，不需要理解数据含义
- **学生不理解双层架构**（新增） → 用"手机+充电宝"类比："不接充电宝手机也能用，接了充电宝更持久"

---

### 队员3（第1-2天）

**教练动作**：
1. 用5章故事讲育种历史（每章5分钟，共25分钟）
2. 让学生复述每章的核心矛盾
3. 开始背第1段话术（立意部分，30秒）

**验收标准**：
- 能复述5章故事主线
- 能说出5个时间节点
- 第1段话术基本能背出来

**常见问题**：
- 学生记不住5个时间节点 → 用手势辅助（1万年→10年→5年→几个月→2035年？）
- 学生语速太快 → 让学生在"关键数字"前停顿1秒
- 学生说"10000年"没有对比感 → 教学生重音："一万年"（重）vs "10年"（轻快）

## 三、第3-4天：背话术

### 背诵方法（通用）

**三遍法**：
1. 看着稿子念3遍（熟悉内容）
2. 看着稿子背3遍（形成记忆）
3. 不看稿子背3遍（测试记忆）

**计时法**：
- 每段话术有目标时间
- 用手机计时，反复练到时间达标
- 如果超时10秒以上 → 删减非核心句子

**录音法**：
- 学生用手机录音，回放听自己的语速
- 教练听录音，指出需要改进的地方

---

| 话术2：断网自治演示 | 20秒 | □ 背熟 | □ 流畅 |

| 答辩稿第2段：总结 | 30秒 | □ 背熟 | □ 流畅 |

## 四、第5-6天：合练

### 合练步骤

**Step 1：两两配合（30分钟）**
- 队员3 + 队员1：练"立意→硬件"交接
- 队员1 + 队员2：练"硬件→软件"交接
- 队员2 + 队员3：练"软件→总结"交接

**Step 2：三人连贯（30分钟）**
- 按联排脚本走完整3分钟
- 教练计时，记录每个环节的实际时间
- 发现问题，当场纠正

**Step 3：重复练习**
- 每次走完后，指出1-2个需要改进的点
- 再走一遍
- 目标：3分钟内完成，交接无停顿

---

### 合练常见问题 + 解决方案

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 交接时停顿超2秒 | 不知道该谁说了 | 练交接句：每人的最后一句就是下一个人的信号 |
| 队员1 OLED翻页卡住 | 按键没按到位 | 训练"按下去听到滴答声"的力度 |
| 队员1拔WiFi后OLED没反应 | 设备问题 | 提前测试，确保拔线后屏幕切换 |
| 队员2大屏操作手忙脚乱 | 不熟练 | 设置桌面快捷方式，双击打开 |
| 队员3语速太快 | 紧张 | 用手机录音，让学生听自己的语速 |
| 三人站位置混乱 | 没有排练 | 画"舞台站位图"贴在墙上 |
| 总时间超3分钟 | 某个环节太长 | 用计时器，超时的环节删减非核心句子 |

## 五、第7天：全要素联排

### 联排流程

**Round 1：完整走一遍（不计时）**
- 目标：流畅走完，不管时间
- 发现问题，记下来

**Round 2：计时走一遍（3分钟目标）**
- 目标：3分钟内完成
- 如果超时，讨论哪里可以压缩

**Round 3：模拟答辩**
- 教练扮评委，随机问5个问题
- 测试三人的Q&A应对能力
- 重点练"反守为攻必杀句"

**Round 4：最终确认**
- 每人背一遍自己的话术（不卡顿）
- 确认硬件、软件都正常
- 互相鼓励

---

## 六、第8-9天：新增专项训练

### 第8天：队员1 · OLED菜单盲操作 + ADC键盘原理

**训练目标**：不看屏幕也能完成菜单翻页操作

**教练动作**：
1. 让学生看着OLED操作菜单5遍（正常操作）
2. 让学生闭眼操作菜单5遍（盲操作）
3. 教ADC键盘原理：
   - "4个键各串联不同电阻——红330Ω、黄680Ω、绿1.5KΩ、蓝3.3KΩ"
   - "按不同键，电压不同，ESP32用1个ADC引脚就能区分"
   - 让学生画电阻梯原理图（简图即可）

**验收标准**：
- 盲操作菜单翻页+进入+退出，一次过
- 能说出4个电阻值和对应颜色
- 能画简图解释原理

---

### 第8天：队员2 · UART链路 + 双层架构补位

**训练目标**：能脱口说出双层架构金句

**教练动作**：
1. 教UART通信4种消息类型：report/advice/ping/pong
2. 教双层架构金句："ESP32是舱内飞控，树莓派是载荷计算机"
3. 模拟评委追问：
   - "比赛为什么不接Pi？" → "WiFi模式就够了，UART_ENABLED=False"
   - "Pi断联怎么办？" → "30秒超时自动回退本地规则"

**验收标准**：
- 金句脱口而出
- 3个追问能流畅回答

---

### 第8天：队员3 · 双层架构话术

**训练目标**：能完整说出双层架构答辩话术

**教练动作**：
1. 让学生背诵第十五步话术（约40秒）
2. 模拟追问："比赛为什么不接Pi？""UART可靠吗？"

**验收标准**：
- 话术背熟，40秒内流畅说完
- 2个追问能接上

---

### 第9天：合练闭环

**内容**：
- 队员1+2合练：OLED翻页 + 菜单操作（30分钟）
- 全员联排一遍（按联排脚本v2.0，30分钟）

**验收标准**：
- OLED翻页演示流畅
- 双层架构金句全员能补位

## 七、教练注意事项

### ⚠️ 训练中的"雷区"

| 雷区 | 为什么是雷区 | 正确做法 |
|------|-------------|----------|
| **不要让学生理解代码** | 零基础7-9天不可能理解，只会增加挫败感 | 只要求能说、能演示 |
| **不要追求完美** | 完美主义会导致紧张和忘词 | 追求"流畅"而非"完美" |
| **不要同时训三个人** | 一个人的问题会影响其他人 | 分头训，最后合练 |
| **不要在合练时打断** | 打断会破坏节奏和信心 | 走完后一起复盘 |
| **不要批评"说得不好"** | 负面反馈会让零基础学生放弃 | 说"这一遍比上一遍好很多" |
| **OLED按键力度**（新增） | 学生可能轻轻碰导致不触发 | 教"按下去听到滴答声才算" |

### ✅ 训练中的"加分动作"

| 加分动作 | 为什么有效 | 怎么做 |
|----------|-----------|--------|
| **录音回放** | 学生听到自己的声音，会主动改进 | 每次背话术都录音 |
| **录像回放** | 看到自己的动作，会主动调整 | 合练时录像 |
| **互相夸奖** | 团队凝聚力，减少紧张 | 每次走完后，三人互相说一个优点 |
| **模拟评委夸奖** | 提升信心 | 教练扮评委，故意问简单问题让学生答对 |
| **提前结束** | 保持兴趣 | 如果提前达标，可以提前下课 |
