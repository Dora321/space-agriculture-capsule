# 实机验收清单

用于比赛、答辩或课堂展示前的最后一轮硬件在环检查。建议每次更换 ESP32、传感器、WiFi、服务器地址或水泵供电后重新跑一遍。

## 1. 准备

- ESP32 已烧录 MicroPython，串口可通过 `mpremote` 连接。
- `esp32_firmware/config.py` 已由 `config.py.example` 复制并填好 WiFi、Dashboard、AI 代理地址。
- 已上传 `plants.json`、`main.py` 以及所有 `*_runtime.py` 模块。
- 12V 水泵先断开真实水路，第一次验收只观察继电器吸合和电机转动。
- WS2812 灯条上电先做整条红→黄→绿自检，确认 11 颗灯珠全亮。
- Dashboard 服务已启动，浏览器能打开实时大屏。

## 7. 树莓派双层 UART 模式现场检查（如启用 ESP32→Pi→大屏）

> 双层模式把链路拉长，多了 UART 线、Pi、Pi 联网三个单点。每项都是 2026-05-30 实机验收踩出来的坑。

| 检查项 | 操作 | 期望结果 | 记录 |
| --- | --- | --- | --- |
| 接线三线齐全 + 交叉 | 检查 ESP32 GPIO17→Pi pin10(RXD)、GPIO16←Pi pin8(TXD)、**GND↔GND pin6 共地** | 三根都插到底，无松动 |  |
| **共地最关键** | 轻晃/轻扯线束 | 仍稳；共地一松 Pi 的 RX 悬空，只读到 `0xFF` 噪声、报告全丢 |  |
| 串口连通性自检 | Pi 上只读嗅探 8s（或看网关日志） | 看到**干净 JSON `report`**，不是 `0xFF` 噪声、可打印率应 ~96% |  |
| 内核控制台已释放 | `ls -l /dev/ttyS0` | `crw-rw---- root dialout`（非 600/root）；`console=serial0` 已从 cmdline 移除、getty disabled。Pi 若重刷系统要重查 |  |
| 看门狗不误杀 | 看 `hw-watchdog` 日志无杀网关 | 网关启动参数**不要含 `board` 子串**；dashboard URL 走 `SPACEFARM_DASHBOARD` 环境变量，别用 `--dashboard` 写进命令行 |  |
| 开机自启 | 重启 Pi | `spacefarm-gateway.service` 自动 active、`NRestarts` 不增长 |  |
| 云端大屏转发 | 现场网络 `curl http://43.156.68.157:8790/api/state` | `"live": true` 且数据新鲜（Pi 的 IP 在 DHCP 下可能变，本地访问先确认） |  |
| ESP32 报告决策源 | 网关 `--auto-advice` 跑一会 | ESP32 `report` 的 `ai_src` 变 `pi`（证明真采用了 Pi 建议） |  |

## 8. 开演前 30 分钟 go/no-go 终检（今天的血泪三条）

- [ ] **【血泪1·接线】** 关键杜邦线（尤其**共地**）已焊死 / 锁扣 / 热熔胶应力释放；震一震、搬一下仍稳。**当场嗅探看到干净 report 才算过**，看到 `0xFF` 立刻重插——今天这根地线松了两次。
- [ ] **【血泪2·选择页】** 开机后**先按绿键走完植物 + 天数选择**，确认进入主循环（OLED 开始三页轮播）才会发报/才有大屏数据。把这一步写进演示脚本第一行——今天就卡在没点选择页。
- [ ] **【血泪3·Pi 隐患】**（双层时）看门狗参数、`console=serial0` 已移除、自启 enabled 三项已按 §7 确认；演示机上**不临时改网关启动参数**。
- [ ] **【供电】** ESP32 USB、Pi、12V 三路电源插牢，自备插排；带**手机热点 + 充电宝兜底**，不赌场馆 WiFi。
- [ ] **【降级演练】** 当场拔网线/断热点，确认**本地规则照跑、OLED/WS2812 照动**——这是"深空自治"叙事高光，也是网络挂了的保命底牌。
- [ ] **【零依赖核心】** 确认只要有电，ESP32 `sense→decide→act→OLED→WS2812` 能完全独立跑通。**把它当主菜，Pi/云/AI 当配菜**：配菜挂了主菜照上。

## 9. 通过标准

- 连续运行 30 分钟无异常重启。
- 传感器离线、WiFi 断线、AI 失败均能降级运行。
- 12V 水泵不会超过配置的安全时长和频次（单次 ≤ 60s、每小时 ≤ 12 次、温度异常时跳过）。
- Dashboard 能展示从采样到决策再到执行的完整闭环。
- 串口日志、OLED、大屏三处信息相互一致。
- （双层模式）UART 链路嗅探为干净 report、`spacefarm-gateway.service` 开机自启、云端大屏 `live:true`；拔掉 Pi 联网后 ESP32 本地闭环仍正常。
