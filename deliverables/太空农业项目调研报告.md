## 二、🛰️ 在轨真实项目

### 2.1 NASA Veggie 蔬菜生产系统

| 属性 | 详情 |
|------|------|
| **全称** | Vegetable Production System (Veggie) |
| **运营方** | NASA Kennedy Space Center |
| **部署位置** | 国际空间站（ISS） |
| **启动时间** | 2014年首次启用 |
| **硬件尺寸** | 约 29 × 37 cm 种植面积 |
| **核心机制** | "种子枕头"（Seed Pillows）— 特殊粘土基质 + 控释肥料 |
| **光照系统** | 红/蓝/绿三色 LED |
| **自动化程度** | 半自动（需宇航员浇水、监控） |

**已种植作物**：
- 生菜（Outredgeous Red、Dragoon）
- 芥菜（Wasabi mustard greens）
- 羽衣甘蓝（Red Russian Kale）
- 百日菊（Zinnia，首次在轨开花植物）
- 白菜（Tokyo Bekana）
- 薄荷

**最新进展**（VEG-03 MNO，2025年8月）：
- 宇航员可从种子库**自主选择**种植品种
- 首次允许在受控植物生长室**之外**种植，利用新型居住空间
- 部分收获直接食用，部分冷冻带回地球分析

**与本项目对比**：

| 对比维度 | NASA Veggie | 本项目（太空农业种植舱） |
|---------|-------------|----------------------|
| 运行环境 | 微重力（ISS） | 地球1g重力 |
| 决策方式 | 宇航员手动 + 地面指令 | 本地规则引擎 + AI 双决策 |
| 灌溉方式 | 种子枕头毛细管供水 | 水泵+继电器主动灌溉 |
| 传感器 | 基本环境参数 | DHT11+土壤湿度+光照+ADC |
| 离线能力 | 依赖地面通信 | 本地规则引擎完全离线运行 |
| 种植规模 | 实验级（几株） | 教育级（1-2盆） |

### 2.2 NASA APH 先进植物栖息地

| 属性 | 详情 |
|------|------|
| **全称** | Advanced Plant Habitat (APH) |
| **运营方** | NASA |
| **部署位置** | 国际空间站 EXPRESS 机架 |
| **规模** | ISS 上**最大**的全自动植物生长设施（约迷你冰箱大小） |
| **自动化程度** | **全自动化**闭环生命支持系统 |
| **实验时长** | 最长 135 天 |
| **生长基质** | 多孔粘土基质 + 控释肥料 |
| **宇航员参与** | 极少（仅安装、加水等维护） |

**已进行实验**：

| 实验编号 | 研究内容 | 关键发现 |
|---------|---------|---------|
| Plant Habitat-01 | 太空vs地球植物遗传学/代谢/光合比较 | 微重力影响基因表达 |
| Plant Habitat-02 | 萝卜 — 代谢、风味、矿物质吸收 | 太空萝卜风味不同于地球 |
| Plant Habitat-03 | 表观遗传是否可传递给下一代 | 首次研究太空表观遗传 |
| Plant Habitat-04 | 新墨西哥哈奇青辣椒 | APH 能力展示 |
| Plant Habitat-06 | 微重力对植物防御反应的影响 | 进行中 |

**与本项目对比**：APH 是**全自动化闭环系统**，与本项目的"本地规则引擎自动 + AI 优化"思路最接近。但 APH 是实验级高端设备，本项目是教育级入门系统。

**与本项目对比**：

### 2.4 Artemis III LEAF 实验（即将执行）

| 属性 | 详情 |
|------|------|
| **全称** | Lunar Effects on Agricultural Flora |
| **运营方** | NASA |
| **计划时间** | 2026年（随 Artemis 3 登月） |
| **历史意义** | **人类首次在月球表面种植植物** |
| **核心目标** | 观察太空辐射+部分重力下植物的光合作用、生长、应激反应 |

**科学问题**：
1. 植物在月球辐射+1/6g下的光合作用如何？
2. 植物的系统性应激反应是什么？
3. 月球种植的植物能否用于人类营养和生命支持？

> 📌 注意：中国嫦娥4号已于2019年在月球背面种植棉花，但那是月球背面的着陆器内部，非宇航员部署。

### 3.1 EDEN ISS（南极温室模拟）

| 属性 | 详情 |
|------|------|
| **全称** | EDEN ISS — Ground Demonstration of Plant Cultivation |
| **主导机构** | DLR（德国航天中心） |
| **部署位置** | 南极诺伊迈尔三号站（Neumayer Station III） |
| **资金来源** | 欧盟 Horizon 2020 |
| **项目周期** | 2015-2019（主项目），延展至2021+ |
| **联合体** | 14个机构（德/奥/意/加/荷/瑞典/爱尔兰/美） |

**核心技术**：
1. 先进营养输送系统
2. 高性能 LED 照明
3. 生物检测与去污系统
4. 食品质量与安全程序

**种植作物**：生菜、黄瓜、番茄、辣椒、芥菜、罗勒、草本植物

**关键成就**：
- 9个月南极越冬期间，为科考队员**全年供应新鲜蔬菜**
- NASA 2021年派驻客座科学家 Jess Bunchek，进行联合实验
- 验证了"封闭/半封闭系统中安全食品生产"全流程

**与本项目对比**：

| 对比维度 | EDEN ISS | 本项目 |
|---------|----------|--------|
| 规模 | 集装箱温室 | 桌面级 |
| 环境 | 南极极端环境 | 室内普通环境 |

| 共同点 | **封闭系统+受控环境+LED光照+营养精准输送** | 同左 |

> ⭐ **答辩亮点**：EDEN ISS 证明"地球极端环境模拟→验证太空种植技术"这条路是可行的，我们这个项目就是这条路上的**最小可行原型**。

### 3.2 MELiSSA（欧空局闭环生命保障系统）

| 属性 | 详情 |
|------|------|
| **全称** | Micro-Ecological Life Support System Alternative |
| **主导机构** | ESA（欧洲航天局） |
| **启动时间** | 1989年（概念始于1987-1988） |
| **运行时长** | **超过30年** |
| **目标** | 开发闭环生物再生生命保障系统：废弃物→食物+水+氧气 |
| **合作网络** | ~50个组织，15个MoU合作伙伴 |

**五隔室架构**：

| 隔室 | 功能 | 对应生态角色 |
|------|------|-------------|
| Compartment I | 废物降解与液化 | 分解者 |
| Compartment II | 硝化（氨→硝酸盐） | 硝化细菌 |
| Compartment III | 光生物反应器（光合微生物→O₂+食物） | 初级生产者（微藻） |
| Compartment IV | 高等植物栽培（食物+空气净化+水净化） | 初级生产者（高等植物） |
| Compartment V | 消费者（航天员） | 消费者 |

**与本项目对比**：

| 对比维度 | MELiSSA | 本项目 |
|---------|---------|--------|
| 系统闭环度 | 全闭环（废物→食物→废物循环） | 半闭环（仅监测+灌溉） |
| 植物角色 | 食物+O₂+净水+CO₂吸收 | 教育演示 |
| AI应用 | 建模与控制层 | 决策引擎层 |
| 技术层级 | 工程级 | 教育级 |

> ⭐ **答辩亮点**：MELiSSA 的 Compartment IV（高等植物栽培）就是我们的"放大版"。我们的项目验证的是：在最小的硬件成本下，能否实现类似的环境监测-决策-执行闭环。

## 四、🌾 中国航天育种工程

| 属性 | 详情 |
|------|------|
| **起点** | 1987年（第九颗返回式卫星首次搭载水稻、辣椒种子） |
| **原理** | 利用太空宇宙射线、微重力、高真空诱变基因变异 |
| **育种周期** | 上天1-27天 → 地面选育8-10年 → 审定品种 |
| **变异概率** | 千分之几到万分之几 |
| **审定品种** | 260+个主粮品种、上百个蔬果林草花卉品种 |
| **年增产粮食** | 20+亿公斤 |
| **经济效益** | **逾千亿元** |
| **当前阶段** | "空间站时代航天育种" |

**标志性成果**：

| 品种 | 成果 |
|------|------|
| 太空莲36号 | 推广2000万亩，占全国白莲80%，亩产翻番 |
| 鲁原502小麦 | 累计推广1亿亩，航天育种中**面积最大**品种 |
| Ⅱ优航1号水稻 | 国审超级稻，保持再生稻百亩单产**世界纪录** |
| 航麦802/106 | 高产优质抗病小麦 |

> ⭐ **答辩关键区别**：航天育种是"上天诱变→地面选育"，种子发生的是**基因变异**；本项目的"太空农业种植舱"是"在轨种植→在轨食用"，解决的是**在太空环境中持续生产食物**的问题。两者是完全不同的技术方向，互为补充。

## 七、本项目的独特定位

### 7.1 在整个太空农业版图中的位置

### 7.2 三个不可替代的差异化优势

1. **教育级唯一的双决策系统**：在所有教育类项目中，本项目是唯一实现了"本地规则+远程AI"双决策引擎的，这个架构直接对标 NASA APH 的全自动化思路
2. **离线优先（Offline-first）架构**：所有在轨项目都无法依赖实时通信（通信窗口有限），本项目从设计之初就遵循"离线优先"原则，这是太空农业的真实需求
3. **8作物精细模型**：不是简单的"土壤干了就浇水"，而是8种作物×多生长阶段×NPK/光/温/水多维度的精细管理模型，这个颗粒度在同类教育项目中不存在

### 7.3 答辩核心叙事

> "我们的项目不是又一个智能花盆。它是对NASA APH全自动化种植舱和天宫菜园第二代自动化系统的**教育级最小可行复刻**——用ESP32+MicroPython，以不到200元的硬件成本，验证了'离线规则引擎+远程AI优化'这条太空农业自动化路径的可行性。"
