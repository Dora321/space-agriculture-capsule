## 3. 系统架构图文案

### ESP32 舱内飞控：只要有电就能跑

**感知 SENSE**

- 土壤湿度 ADC
- 环境光照 ADC
- DHT 温湿度
- 旋转编码器 + OLED 菜单

**决策 THINK**

- 本地规则引擎常驻兜底
- 采纳树莓派 DeepSeek advice
- 所有外部建议先过本地安全护栏

**执行 ACT**

- 12V 水泵精量浇水
- 12V COB 补光灯
- WS2812 决策信号广播
- OLED 三页状态显示

**安全 SAFE**

- 单次动作时长上限
- 每小时动作次数限制
- 高低温跳过危险动作
- 传感器异常自动降级

### 树莓派载荷计算机：断了只是变笨

- UART 接收 ESP32 report
- 调 DeepSeek 生成 advice
- 转发 Web 地面站大屏
- AI / 网络失败时不下发 advice，ESP32 自动回到本地规则

### 两层降级链

DeepSeek 精细判断 → ESP32 本地规则保命兜底

## 4. 五大核心亮点

### 1. 深空自治架构

ESP32 负责“作物会不会死”的飞控任务；树莓派负责联网、AI 和大屏。断网时系统不是停机，而是自动切回本地规则。

### 2. Decision Plane / Action Plane 分离

AI 和规则可以输出缺水、缺光、高温、缺肥等多维诊断信号；水泵和补光灯只执行有硬件支撑的动作，其余信号由 WS2812 灯条广播。

### 3. 8 种作物参数库

内置生菜、小白菜、菠菜、韭菜、番茄、辣椒、黄瓜、茄子。现场用旋转编码器切换，阈值、生长阶段和策略自动适配。

### 4. 全生长周期数据闭环

系统记录土壤、光照、温湿度、天数、生长阶段、动作次数和 AI 育种观察，为“哪一株更值得留种”积累时间序列证据。

### 5. 地面站实时遥测

Web 大屏展示 LIVE 状态、传感器仪表、动作链路、DeepSeek 原因、决策信号和育种观察，模拟未来地面飞控中心。
