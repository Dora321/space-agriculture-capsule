系统链路 感知 SENSE -> 决策 THINK -> 执行 ACT -> 展示 DISPLAY

感知层 SENSE

  • 土壤湿度 ADC
  • 环境光照 ADC
  • DHT11/DHT22 温湿度
  • 3位拨码开关选择作物
现场 8 种核心作物可一键切换
->

决策层 THINK

  • ESP32 + MicroPython 主控循环
  • 本地规则常驻兜底
  • DeepSeek API 或 AI 代理
  • 按种植日期判断生长阶段
阈值事件、明显变化或定时复核时请求 AI
->

执行层 ACT

  • 12V 水泵 + 单路继电器自动浇水
  • WS2812 RGB 灯条 11 颗灯珠（湿度温度计 + 系统状态）
  • 单次 60 秒上限，12 次/小时限频
  • 温度异常 ≥35℃ 或 ≤8℃ 跳过浇水
异常动作强制 idle，传感器离线降级
->

展示层 DISPLAY

  • OLED 三页轮播
  • 动作执行覆盖提示
  • ESP32 POST 实时遥测
  • Web 大屏 LIVE/DEMO 自动切换
120 秒无实时数据时自动回到演示状态

核心亮点 从原型到可演示系统

AI + 本地规则双决策

云端 AI 做综合判断，本地规则保证离线可用；网络或 API 失败时仍可自动养护。

14 种作物参数库

每种作物包含湿度阈值、浇水时长、光照需求与生长阶段表。

生长阶段自适应

根据种植日期判断苗期、营养期、花期、果期或采收期，动态调整水肥策略。

实时数据大屏

仪表盘、趋势曲线、动态植物、当前决策和系统状态同步展示。

全链路安全保护

动作限频、时长上限、传感器离线降级、错误计数和看门狗重启，面向长期无人值守。
