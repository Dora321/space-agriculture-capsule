# 太空农业种植舱项目 (Space Agriculture Growth Chamber Project)

本项目旨在开发一个智能农业种植舱的软硬件一体化解决方案。系统以 ESP32 为核心控制单元，结合多种环境传感器和执行机构（水泵、营养液泵等），能够通过本地预设规则和云端 AI 双重机制，对舱内植物实现智能环境监控与自主养护。

## 📂 项目结构

- [`esp32_firmware/`](./esp32_firmware/)
  ESP32 下位机固件代码目录。包含基于 MicroPython 编写的各个模块：传感器采集、执行器控制、OLED 显示、Wi-Fi 客户端以及 AI 大模型交互客户端。有关固件的详细说明、烧录指南和接线方法，请参阅该目录下的 [README.md](./esp32_firmware/README.md)。

- [`智能种植舱控制器选型报告.md`](./智能种植舱控制器选型报告.md)
  系统硬件设计的详细文档，涵盖了主控芯片、传感器、执行器、显示模块等硬件选型依据，并提供了详细的接口分配表。

## ✨ 核心特性

- **多传感器融合**：支持土壤湿度、环境光照、温湿度等多种环境参数的实时采集。
- **AI 决策与本地规则双擎**：当网络和 API 可用时，可通过接入外部大语言模型（如 DeepSeek）进行动态的环境决策；当离线或 API 故障时，自动降级为本地阈值控制。
- **多植物种类支持**：内置生菜、小白菜、番茄等多种常见农作物的生长阈值和养护参数。
- **安全与兜底机制**：包含传感器断连保护、强制执行超时控制以及异常状态警报，确保种植系统稳定安全运行。
- **实时状态显示**：搭配 OLED 屏幕，直观显示实时数据、当前执行动作及异常状态信息。

## 🚀 快速开始

如果您想直接部署该项目：

1. **查阅硬件选型**：阅读 [`智能种植舱控制器选型报告.md`](./智能种植舱控制器选型报告.md) 了解所需硬件和连接方式。
2. **配置与烧录**：进入 `esp32_firmware/` 目录，按照其 [README.md](./esp32_firmware/README.md) 修改 `config.py` 中的网络与 API 密钥。使用 esptool 烧录 MicroPython 并将 Python 代码上传至 ESP32 开发板。
